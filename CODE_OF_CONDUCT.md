# 贡献者公约 · Code of Conduct

## 我们的承诺

为建设开放、友善的环境，我们承诺：参与本项目的贡献者与维护者均不受骚扰，无论年龄、体型、身体健全与否、民族、性征、性别认同与表达、经验、国籍、外貌、宗教或性取向。

## 我们的标准

有助于创造积极环境的行为包括：

- 使用友善、包容的语言
- 尊重不同的观点与经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员保持同理心

不可接受的行为包括：

- 性化的语言或意象、不受欢迎的献殷勤
- 挑衅、侮辱/贬损的评论、人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可公开他人的私人信息
- 在专业场合可被合理认为不适当的其他行为

## 我们的责任

项目维护者有责任阐明可接受行为的标准，并对不当行为采取恰当、公平的纠正措施。

## 适用范围

本公约适用于本仓库内及公开场合代表本项目时的行为。代表本项目包括使用官方邮箱、社交媒体账号，或在在线/线下活动中担任指定代表。

## 执行

辱骂、骚扰或其他不可接受的行为可向项目团队报告。所有投诉将被审查与调查，并视情况作出被认为必要且适当的回应。维护者有义务对报告者保密。具体执行政策可另行发布。

不遵守本公约者可被暂时或永久取消参与本项目的资格。
